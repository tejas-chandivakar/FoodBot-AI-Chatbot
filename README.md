# FoodBot AI Chatbot 🍽️🤖

An AI-powered smart restaurant chatbot developed using **Rasa NLP**, **Python**, and a professional **Web-based User Interface**. FoodBot automates restaurant ordering and customer interaction through intelligent conversational AI in both **English** and **Hindi** languages.

## 📌 Project Overview

FoodBot is designed to provide an intelligent restaurant automation experience where users can interact naturally with the chatbot for ordering food, managing carts, tracking orders, viewing order history, and getting customer support assistance.

The project demonstrates the practical implementation of:

* Artificial Intelligence (AI)
* Natural Language Processing (NLP)
* Conversational Chatbots
* Web Technologies
* REST API Integration

This project was developed as an MCA Major Project.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/tejas-chandivakar/FoodBot-AI-Chatbot.git
```

### 2️⃣ Open Project

```bash
cd FoodBot-AI-Chatbot
```

### 3️⃣ Install Requirements

```bash
pip install rasa
pip install rasa-sdk
```

---

## ▶️ Run Project

### Start Action Server

```bash
rasa run actions
```

### Train Chatbot

```bash
rasa train
```

### Run Rasa Server

```bash
rasa run --enable-api --cors "*"
```
